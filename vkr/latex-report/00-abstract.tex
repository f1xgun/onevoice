Отчёт по описанию проекта \textbf{OneVoice}~--- мультиагентной системы с гибридной интеграцией (API~+ RPA) для управления цифровым присутствием МСБ. В документе приведены ответы на пункты чек-листа: проблема и целевая аудитория, текущее состояние, аналоги и критерии, предметная область, технологии, данные. Цель~--- разработка программного комплекса; задачи перечислены во введении.
