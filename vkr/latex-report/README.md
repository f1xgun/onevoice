# Отчёт «Описание проекта» (OneVoice)

Краткий отчёт по описанию проекта (3–4 страницы) в формате LaTeX с ответами на вопросы чек-листа из `docs/описание проекта.docx`. Оформление по шаблону ГОСТ 7-32 (класс G7-32).

## Сборка

Требуется **XeLaTeX** (для шрифтов Times New Roman).

```bash
cd vkr/latex-report
make
```

Или вручную:

```bash
xelatex report.tex
xelatex report.tex
```

Результат: `report.pdf`.

## Структура (компактная, 3–4 стр.)

- `report.tex` — главный файл
- `00-title.tex` — титульный лист (название проекта, ФИО)
- `00-abstract.tex` — краткая аннотация
- `12-intro.tex` — введение (цель, отсылка к задачам)
- `10-body-short.tex` — одна глава с разделами: актуальность/цель/задачи, проблема и ЦА, текущее состояние, аналоги и критерии, предметная область, технологии, данные, итог
- `81-biblio.tex` — список источников (подключает `../latex/bibliography.tex`)

## Рисунки

Схема AS-IS в разделе 2 описана текстом; при необходимости рисунок можно сгенерировать из `../latex/images/as-is.mmd` (Mermaid) и положить в `images/as-is.png`, после чего раскомментировать блок с `\includegraphics` в `chapters/02-current-state.tex`.

## Зависимости

Список литературы общий с пояснительной запиской ВКР: `vkr/latex/bibliography.tex`. Стили и класс (G7-32, G2-105, GostBase, preamble) расположены в этой же директории.
