Управление цифровым присутствием является одной из ключевых задач для субъектов малого и среднего бизнеса (далее~--- МСБ) в~условиях цифровой экономики. Распределённость информации о~компании по множеству платформ и каналов, а~также гетерогенность их интеграционных интерфейсов формируют потребность в~автоматизированных инструментах контроля и обновления данных.

Целью данной работы является разработка программного комплекса на основе мультиагентной архитектуры с~гибридной моделью интеграции (API~+ RPA) для управления цифровым присутствием субъектов МСБ.

Для достижения указанной цели необходимо выполнить следующие задачи:

\begin{enumerate}
  \item провести анализ предметной области и~существующих решений;
  \item сформулировать требования к~системе и~предложить инфологическую модель данных;
  \item обосновать выбор технологий и~архитектурных решений;
  \item спроектировать архитектуру мультиагентной системы;
  \item реализовать прототип программного комплекса;
  \item провести тестирование и~оценку эффективности разработанного решения.
\end{enumerate}
