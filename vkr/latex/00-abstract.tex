Выпускная квалификационная работа (далее ВКР) на тему: <<Разработка программного комплекса на основе мультиагентной архитектуры с~гибридной моделью интеграции для управления цифровым присутствием субъектов малого и среднего бизнеса>>.

Цель ВКР~--- разработка и~реализация программного комплекса на основе мультиагентной архитектуры с~гибридной моделью интеграции (API~+ RPA), обеспечивающего автоматизацию управления цифровым присутствием субъектов малого и среднего бизнеса на нескольких внешних площадках.
