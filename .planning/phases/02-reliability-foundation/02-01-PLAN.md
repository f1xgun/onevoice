# PLAN-2.1: NonRetryableError type + withRetry integration

## Summary

Add a `NonRetryableError` struct to `pkg/a2a/protocol.go` that implements the `error` interface with `Is()` and `Unwrap()` support. Integrate it into the Yandex agent's `withRetry` function so permanent failures short-circuit immediately instead of exhausting all retry attempts.

## Tasks

<task id="1">
<title>Add NonRetryableError type to pkg/a2a/protocol.go</title>
<read_first>
- pkg/a2a/protocol.go
- pkg/a2a/agent.go
</read_first>
<action>
Add the following to `pkg/a2a/protocol.go` after the existing `ToolResponse` struct:

```go
// NonRetryableError marks an error as permanent ‚Äî withRetry must not retry it.
type NonRetryableError struct {
	Err error
}

func (e *NonRetryableError) Error() string { return e.Err.Error() }
func (e *NonRetryableError) Unwrap() error { return e.Err }
func (e *NonRetryableError) Is(target error) bool {
	_, ok := target.(*NonRetryableError)
	return ok
}

// NewNonRetryableError wraps err as a permanent failure that should not be retried.
func NewNonRetryableError(err error) *NonRetryableError {
	return &NonRetryableError{Err: err}
}
```

Add `"errors"` to the import if not already present (it is not ‚Äî only `"fmt"` is imported).
</action>

- `cd pkg && go build ./...` exits 0
</acceptance_criteria>
</task>

1. Add `"errors"` and `"github.com/f1xgun/onevoice/pkg/a2a"` to imports.
2. Modify the `withRetry` function: after `lastErr = fn()`, add a check before the sleep:

```go
lastErr = fn()
if lastErr == nil {
    return nil
}
if errors.Is(lastErr, &a2a.NonRetryableError{}) {
    return lastErr
}
if i < maxAttempts-1 {
    time.Sleep(time.Duration(1<<uint(i)) * time.Second)
}
```

This makes withRetry return immediately on permanent errors without retrying.
</action>
