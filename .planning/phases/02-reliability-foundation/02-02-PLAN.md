# PLAN-2.2: Error taxonomy across all agents

## Summary

Apply error classification to all three agent handlers (Telegram, VK, Yandex.Business). Permanent platform errors (invalid token, access denied, invalid parameters) are wrapped in `a2a.NonRetryableError` so the orchestrator's retry logic and the Yandex `withRetry` function skip them immediately. Transient errors (network, timeouts) remain as regular errors for retry eligibility.

## Error Classification Reference

| Category | Behavior | VK Codes | Telegram | Yandex |
|----------|----------|----------|----------|--------|
| Permanent | `NonRetryableError` | 5 (invalid token), 15 (access denied), 100 (invalid param), 113 (invalid user) | 401 Unauthorized, 403 Forbidden | login redirect, session expired |
| Rate-limited | `NonRetryableError` with descriptive message | 6 (too many requests), 9 (flood control) | 429 Too Many Requests | CAPTCHA detection |
| Transient | Normal error (retryable) | All other codes, network errors | Network errors, 5xx | Playwright timeout, network errors |

// classifyVKError wraps permanent VK API errors as NonRetryableError.
// VK error codes: 5=invalid token, 15=access denied, 100=invalid param, 113=invalid user,
// 6=too many requests, 9=flood control (rate-limited, also non-retryable).
func classifyVKError(err error) error {
	var vkErr *vkapi.Error
	if !errors.As(err, &vkErr) {
		return err // network or non-VK error — transient, retryable
	}
	switch vkErr.Code {
	case 5, 15, 100, 113: // permanent
		return a2a.NewNonRetryableError(err)
	case 6, 9: // rate-limited — don't retry, surface to user
		return a2a.NewNonRetryableError(fmt.Errorf("vk rate limit (code %d): %w", vkErr.Code, err))
	default:
		return err // transient
	}
}
```

// classifyTelegramError wraps permanent Telegram API errors as NonRetryableError.
// Checks error message strings since tgbotapi returns errors with descriptions.
func classifyTelegramError(err error) error {
	if err == nil {
		return nil
	}
	msg := err.Error()
	// Permanent: unauthorized, forbidden
	if strings.Contains(msg, "Unauthorized") || strings.Contains(msg, "Forbidden") {
		return a2a.NewNonRetryableError(err)
	}
	// Rate-limited: too many requests — non-retryable, surface to user
	if strings.Contains(msg, "Too Many Requests") || strings.Contains(msg, "retry after") {
		return a2a.NewNonRetryableError(fmt.Errorf("telegram rate limit: %w", err))
	}
	// Chat/channel not found — permanent
	if strings.Contains(msg, "chat not found") || strings.Contains(msg, "Bad Request: chat_id is empty") {
		return a2a.NewNonRetryableError(err)
	}
	return err // transient (network, 5xx, etc.)
}
```

}
