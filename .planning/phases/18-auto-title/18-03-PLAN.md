<objective>
Build the trust-critical persistence primitives for Phase 18:

1. **UpdateTitleIfPending(ctx, id, title)** — atomic conditional Mongo write that
   guards manual renames from titler clobber (TITLE-04, D-08). The filter
   `{_id, title_status: {$in: ["auto_pending", null]}}` matches zero docs when a
   manual rename has flipped status to "manual" mid-flight; the titler call
   becomes a silent no-op surfaced as ErrConversationNotFound.

2. **TransitionToAutoPending(ctx, id)** — atomic conditional flip from
   {auto, null} → "auto_pending" used by Plan 05's regenerate-title endpoint
   (TITLE-09, D-07). Filter-fail → ErrConversationNotFound → handler returns
   409 Conflict with appropriate Russian body.

3. **D-06 plumbing fix (Landmine 7):** existing `Update(ctx, conv)` method's
   `$set` block must include `title_status` — otherwise Plan 05's
   `conversation.TitleStatus = "manual"` handler change is a silent no-op.

4. **D-08a compound index** registered idempotently at API startup via the
   existing `EnsureXxxIndexes` pattern. Filled into the slot Plan 02 reserved
   in `services/api/cmd/main.go`.

Purpose: Without these primitives, the entire phase's trust contract breaks.
Manual renames stop being sovereign; titler clobbers user intent (the bug
PITFALLS §12 flags as the trust-critical failure mode).

       ```go
       // UpdateTitleIfPending — Phase 18 / TITLE-04 / D-08.
       // Mongo UpdateOne with $in filter on title_status; manual renames mid-flight
       // match zero documents and the titler write becomes a silent no-op.
       func (r *conversationRepository) UpdateTitleIfPending(ctx context.Context, id, title string) error {
           filter := bson.M{
               "_id": id,
               "title_status": bson.M{
                   "$in": []interface{}{domain.TitleStatusAutoPending, nil},
               },
           }
           update := bson.M{
               "$set": bson.M{
                   "title":        title,
                   "title_status": domain.TitleStatusAuto,
                   "updated_at":   time.Now(),
               },
           }
           result, err := r.collection.UpdateOne(ctx, filter, update)
           if err != nil {
               return fmt.Errorf("update title if pending: %w", err)
           }
           if result.MatchedCount == 0 {
               return domain.ErrConversationNotFound
           }
           return nil
       }

       // TransitionToAutoPending — Phase 18 / TITLE-09 / D-07.
       ```

       ```go
       // EnsureConversationIndexes creates the Phase 18 sidebar / atomic-update
       // index idempotently at API startup. The compound (user_id, business_id,
       // title_status) supports (a) fast UpdateTitleIfPending lookups for the
       // auto-titler and (b) sidebar list queries that filter by user/business
       // and surface auto_pending rows distinctly. Safe to call on every boot —
       // Mongo's CreateMany silently succeeds on existing matching specs.
       func EnsureConversationIndexes(ctx context.Context, db *mongo.Database) error {
           coll := db.Collection("conversations")
           models := []mongo.IndexModel{
               {
                   Keys: bson.D{
                       {Key: "user_id", Value: 1},
                       {Key: "business_id", Value: 1},
                       {Key: "title_status", Value: 1},
                   },
                   Options: options.Index().SetName("conversations_user_biz_title_status"),
               },
           }
           if _, err := coll.Indexes().CreateMany(ctx, models); err != nil {
               if mongo.IsDuplicateKeyError(err) {
                   return nil
               }
               return fmt.Errorf("ensure conversation indexes: %w", err)
           }
           return nil
       }
       ```
