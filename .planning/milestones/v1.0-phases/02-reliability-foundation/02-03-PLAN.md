# PLAN-2.3: Graceful shutdown for all services

## Summary

Add graceful shutdown to all services: HTTP server stop, NATS drain, in-flight request completion, and ordered resource cleanup. The A2A Agent base gets a `sync.WaitGroup` to track goroutines and a `Stop()` method for drain. Each service's `cmd/main.go` is updated to use signal handling, configurable timeout, and correct shutdown ordering.

<title>Add WaitGroup and Stop() to pkg/a2a Agent</title>
<read_first>
- pkg/a2a/agent.go
- pkg/a2a/nats_transport.go
</read_first>
<action>
In `pkg/a2a/agent.go`:

1. Add `"sync"` to imports.
2. Add `wg sync.WaitGroup` field to the `Agent` struct:

```go
type Agent struct {
	id        AgentID
	transport Transport
	handler   Handler
	wg        sync.WaitGroup
}
```

3. In the `Start` method's subscription callback, wrap the goroutine with WaitGroup tracking:

```go
return a.transport.Subscribe(subject, func(subj, reply string, data []byte) {
    a.wg.Add(1)
    go func() {
        defer a.wg.Done()
        a.handle(ctx, reply, data)
    }()
})
```

4. Add a `Stop` method that waits for in-flight handlers to complete:

```go
// Stop waits for all in-flight message handlers to complete.
// It should be called after the transport is closed/drained to ensure
// no new messages arrive while waiting.
func (a *Agent) Stop() {
	a.wg.Wait()
}
```
</action>

<title>Add graceful shutdown to orchestrator</title>
<read_first>
- services/orchestrator/cmd/main.go

</read_first>
<action>
In `services/orchestrator/cmd/main.go`:

1. Add imports: `"os/signal"`, `"syscall"`.
2. Add `SHUTDOWN_TIMEOUT` to config (default 30s) if not already present. In `services/orchestrator/internal/config/config.go`, add:

```go
ShutdownTimeout time.Duration
```

Parse from env: `SHUTDOWN_TIMEOUT` with default `30s` using `time.ParseDuration`.

3. Replace the blocking `srv.ListenAndServe()` at the end of `run()` with signal-based shutdown:

```go
// Start server in goroutine
errCh := make(chan error, 1)
go func() {
    log.Info("orchestrator listening", "addr", addr)
    if err := srv.ListenAndServe(); err != nil && err != http.ErrServerClosed {
        errCh <- err
    }
}()

// Wait for signal
quit := make(chan os.Signal, 1)
signal.Notify(quit, syscall.SIGINT, syscall.SIGTERM)

select {
case <-quit:
    log.Info("shutting down orchestrator")
case err := <-errCh:
    return fmt.Errorf("server error: %w", err)
}

shutdownCtx, cancel := context.WithTimeout(context.Background(), cfg.ShutdownTimeout)
defer cancel()

// 1. Stop HTTP server (drains active SSE connections)
if err := srv.Shutdown(shutdownCtx); err != nil {
    log.Error("http shutdown error", "error", err)
}

// 2. Drain NATS
if nc != nil {
    _ = nc.Drain()
}

log.Info("orchestrator stopped")
return nil
```

4. Add `"context"` and `"os"` to imports if not already present.
</action>
