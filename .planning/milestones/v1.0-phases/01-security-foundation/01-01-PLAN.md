## Tasks

<task id="1-01-01">
<title>Create security headers middleware</title>
<read_first>
- services/api/internal/middleware/auth.go
- services/api/internal/middleware/ratelimit.go
- services/api/internal/router/router.go
</read_first>
<action>
Create new file `services/api/internal/middleware/security.go` with package `middleware`.

Define function:

```go
func SecurityHeaders() func(http.Handler) http.Handler
```

The middleware must set these headers on every response before calling `next.ServeHTTP(w, r)`:

- `X-Content-Type-Options` = `nosniff`
- `X-Frame-Options` = `DENY`
- `Content-Security-Policy` = `default-src 'self'; script-src 'self' 'unsafe-inline'; style-src 'self' 'unsafe-inline'; img-src 'self' data: https:; connect-src 'self'`
- `Referrer-Policy` = `strict-origin-when-cross-origin`
- `Permissions-Policy` = `camera=(), microphone=()`

Import only `net/http`.
</action>
<acceptance_criteria>
- File `services/api/internal/middleware/security.go` exists with `package middleware`
- File contains `func SecurityHeaders() func(http.Handler) http.Handler`
- grep for `X-Content-Type-Options` in security.go returns `nosniff`
- grep for `X-Frame-Options` in security.go returns `DENY`
- grep for `Content-Security-Policy` in security.go returns `default-src 'self'`
- grep for `Referrer-Policy` in security.go returns `strict-origin-when-cross-origin`
- grep for `Permissions-Policy` in security.go returns `camera=(), microphone=()`
- `cd services/api && GOWORK=off go build ./...` succeeds
</acceptance_criteria>
</task>

<task id="1-01-02">
<title>Wire security headers middleware into router</title>
<read_first>
- services/api/internal/router/router.go
- services/api/internal/middleware/security.go
</read_first>
<action>
In `services/api/internal/router/router.go`, inside the `Setup` function, add `r.Use(middleware.SecurityHeaders())` immediately AFTER the `r.Use(cors.Handler(...))` block (after the closing `}))` on line 45) and BEFORE the `r.Route("/api/v1", ...)` block.

The line to add is:
```go
r.Use(middleware.SecurityHeaders())
```

This ensures CORS preflight is handled first, then security headers are applied to all responses.
</action>
